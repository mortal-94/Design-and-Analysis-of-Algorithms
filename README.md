## 三符号：$O、\Omega、\Theta$

$O$ ：渐进上界

$\Omega$ ：渐进下界

$\Theta$ ：渐进紧确界

严格数学定义略。

三种渐进关系用于描述算法的时间和空间复杂性。

```
O(1)（常数）<O(log n) （对数）<O(n) （线性）<O(n log n) （对数线性）<O(nᵏ) （多项式）<O(kⁿ)（指数）<O(n!) （阶乘）
```
